DeadPunk OS is a browser-based fake operating system that serves as MaRCoRe's personal portfolio and project hub. It's not a real OS and it's not trying to be one — it's a creative presentation layer for projects, experiments, and ideas.

## What It Is

DeadPunk OS runs entirely in the browser with no install and no server. It's designed as a portfolio hub rather than a functional operating system — a deliberate creative choice that reflects the personality of the work behind it.

- Runs entirely in the browser — no install, no server
- Designed as a portfolio hub, not a functional operating system
- Reflects the "avant-garde" personality of the projects behind it

  The best way to understand DeadPunk OS is to just open it and explore.
</Tip>
